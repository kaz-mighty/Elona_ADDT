# Elona ADDT DLL

## 概要
Elona ヴァリアントの一つであるADDTのDLLソースコードです。
オリジナルは[Elona板(餅)のスレッド](https://jbbs.shitaraba.net/bbs/read.cgi/game/45610/1350133364/l50)にて別の方が制作されていました。


## 開発環境
オリジナルはVS2013で制作されたようです。
ここではVS2017で制作しています。

## ライセンス
zlib/libpngライセンス
(オリジナルでは

>自作騙りしなければ基本的にADDTは転載●再配布等おkです

とだけ発言されていました。)
